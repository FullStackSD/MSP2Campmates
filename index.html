About

A camping trip needs some planning even for an experienced adventurer. Campmates is a checklist for campers to pack & plan before they set off on an adventure in the great outdoors. We want you to adventure more and worry less.

We have been part of planning camping trips for outdoor enthusiasts all over the world including Cape Town, Sydney and Dublin. We look forward to being part of your journey.

Let’s get you started
There are a lot of complex things in the world, this doesn’t have to be one of them. Let’s break it down for you.
  1. Select the item you have packed or prepared from the pre-prepared list or add your own item by typing in the space provided. If you want to create your own item, go to the "Own List" category and click or tap on the “+” button.
  2. Select the "Checked" option next to each item you have packed. It will then move to the "Already Packed" list.
  3. Select the "Buy” option next to each item if you still need to buy it. You then have the option to select the "Check" button once you have bought and packed the item.
  4. Find an online or physical camping store to purchase what you need. You can interact with the map and select the marker on the map to see store information.
  5. Add a review to give us feedback on how you use Campmates.
